15. Causal Inference: Different Backends (SB for PS, Varied for Outcome)

Causal Inference: Different Backends for PS (SB) vs Outcome

Goal: Apply Stick-Breaking for PS with varied outcome backends for fast, efficient causal inference.

Use Case: When PS model is simpler, outcome requires more flexibility (or vice versa).

Data Setup

set.seed(42)
n <- 200

X <- cbind(
  x1 = rnorm(n, mean = 0, sd = 1),
  x2 = rnorm(n, mean = 0, sd = 1)
)

T <- rbinom(n, size = 1, prob = 0.5)

y <- ifelse(
  T == 1,
  rnorm(n, mean = 3 + 0.5 * X[, 1], sd = 1.2),
  rnorm(n, mean = 1 + 0.3 * X[, 1], sd = 1.0)
)

cat("Sample size:", n, "\n")

Causal Models: SB PS with Varied Outcome Backends

SB PS + CRP Outcome

bundle_sb_crp <- build_causal_bundle(
  y = y,
  T = T,
  X = X,
  kernel = "normal",
  backend = "crp",           # Outcome: CRP (flexible)
  PS = "logit",              # PS: via SB (fixed components)
  GPD = FALSE,
  Kmax = 10,
  mcmc = list(niter = 1000, nburnin = 200, nchains = 1)
)

fit_sb_crp <- run_mcmc_bundle_manual(bundle_sb_crp)

ate_sb_crp <- coef(fit_sb_crp)
cat("=== SB PS + CRP Outcome ===\n")
cat("ATE:", ate_sb_crp, "\n")

SB PS + SB Outcome

bundle_sb_sb <- build_causal_bundle(

  T = T,
  X = X,
  kernel = "normal",
