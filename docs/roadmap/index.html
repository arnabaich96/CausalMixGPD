Function-by-level map

Key entrypoints grouped by where you use them
High-level only
Level 0 — Setup and data
  • init_kernel_registry()
    Initialize / refresh mixture kernel registry.
  • get_kernel_registry()
    Inspect available kernels and parameterization.
  • get_tail_registry()
    Inspect GPD tail parameterization.
  • kernel_support_table()
    Which kernels support GPD, regression, backends, etc.
  • sim_bulk_tail()
    Simulate bulk-plus-tail outcomes for demos/tests.
  • sim_causal_qte()
    Simulate (Y, A, X) for causal QTE examples.
  • sim_survival_tail()
    Simulate survival-style tail-focused examples.
Level 1 — Specify and build
  • build_nimble_bundle()
    Standard builder for unconditional/conditional models.
  • build_causal_bundle()
    Creates treated/control outcome bundles + optional PS design.
Level 2 — Run MCMC
  • run_mcmc_bundle_manual()
    Run posterior sampling for a standard bundle.
  • run_mcmc_causal()
    Run MCMC per arm (+ optional PS model) and return causal fit.
Level 3 — Inspect and diagnose
  • print()
    Concise object view: dimensions, options, flags.
  • summary()
    Posterior summaries for fits; structural summary for bundles.
  • plot()
    Trace/density diagnostics and prediction plots (as implemented).
  • params()
    Structured extraction of posterior draws.
  • check_glue_validity()
    Black-box checks for bulk–tail (GPD) glue validity.
Level 4 — Extract and predict
  • predict()
    Density / survival / quantile / mean / rmean / samples, with intervals.
  • fitted()
    In-sample fitted values for conditional models.
  • residuals()
    Raw residuals and PIT residuals (plugin/bayesian modes).
Level 5 — Causal (optional)
  • qte()
    Conditional quantile treatment effects (draw-wise differences).
  • ate()
    Mean ATE (may be infinite if ξ≥1 has posterior mass).
  • ate_rmean()
    Restricted-mean ATE up to a finite cutoff.
  • predict()
    Applied to causal fit: arm-specific predictions + effect assembly.
