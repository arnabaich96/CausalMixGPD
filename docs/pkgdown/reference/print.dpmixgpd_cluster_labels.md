# Print cluster labels

Compact display for a representative clustering.

## Usage

``` r
# S3 method for class 'dpmixgpd_cluster_labels'
print(x, ...)
```

## Arguments

- x:

  Cluster labels object.

- ...:

  Unused.

## Value

`x`, invisibly.
