# Summarize a fitted causal model

`summary.causalmixgpd_causal_fit()` routes the user to the fitted PS
block, treated/control fits, and downstream prediction/effect methods.

## Usage

``` r
# S3 method for class 'causalmixgpd_causal_fit'
summary(object, ...)
```

## Arguments

- object:

  A `"causalmixgpd_causal_fit"` object.

- ...:

  Unused.

## Value

The input object (invisibly).
