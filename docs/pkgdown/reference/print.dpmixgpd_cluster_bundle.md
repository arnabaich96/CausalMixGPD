# Print a cluster bundle

Compact display for a `dpmixgpd_cluster_bundle` before MCMC is run.

## Usage

``` r
# S3 method for class 'dpmixgpd_cluster_bundle'
print(x, ...)
```

## Arguments

- x:

  A cluster bundle.

- ...:

  Unused.

## Value

`x`, invisibly.
