# Print a cluster posterior similarity matrix

Compact display for a posterior similarity matrix.

## Usage

``` r
# S3 method for class 'dpmixgpd_cluster_psm'
print(x, ...)
```

## Arguments

- x:

  Cluster PSM object.

- ...:

  Unused.

## Value

`x`, invisibly.
