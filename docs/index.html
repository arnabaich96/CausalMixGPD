Bayesian semiparametric modeling for heavy tails

DPmixGPD

Flexible bulk + tail modeling that combines Dirichlet-process mixtures for the bulk with Generalized Pareto tails. Built for reproducible workflows: kernels, backends, MCMC, diagnostics, prediction, and causal/survival extensions.

DPM bulk GPD tail CRP / SB Unconditional & Conditional Causal & Survival
Start with Cookbook Browse Reference

What it does

Bulk modeling via DPM kernels and tail modeling via GPD, with consistent tools for building, running, and extracting results.
