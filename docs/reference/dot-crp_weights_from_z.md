# Compute CRP weights from a vector of allocations z

Compute CRP weights from a vector of allocations z

## Usage

``` r
.crp_weights_from_z(z, Kmax)
```

## Arguments

- z:

  Integer vector of length N with values in 1:Kmax.

- Kmax:

  Integer.

## Value

Numeric vector of length Kmax summing to 1.
