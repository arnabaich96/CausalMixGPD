# Kernel support matrix Returns a data frame summarizing each kernel's supported features.

Kernel support matrix Returns a data frame summarizing each kernel's
supported features.

## Usage

``` r
kernel_support_table(round = TRUE)
```

## Arguments

- round:

  Logical; `TRUE` to replace logical values with symbols.

## Value

data.frame with columns `kernel`, `gpd`, `covariates`, `sb`, `crp`.
